# Design Memory: Kaizen

## Simplified Token System (Refined 2026-05-06)

### Core Colors (Light)
- **Background:** `#f6f8f4` (Soft Mint Grey)
- **Surface:** `#ffffff` (Pure White)
- **Surface Secondary:** `#eef2ed` (Soft Sage)
- **Primary:** `#56c87b` (Kaizen Green)
- **Text Primary:** `#1f2a21` (Deep Forest)
- **Text Secondary:** `#556257` (Sage)

### Typography Decisions
- **System:** Inter (Sans-serif)
- **Main Headers:** Bold (700) with `tracking-tighter`, Uppercase.
- **Card Titles:** Bold (700) Uppercase with `tracking-tight`.
- **Labels:** Bold (700) Uppercase with `widest` tracking.
- **Body:** Medium (500) or Bold (700) for high-impact numbers.

### Component Patterns
- **Bento Card:** `rounded-2xl p-5 md:p-6 border border-border-subtle shadow-sm`
- **Action Button:** `rounded-xl px-4 py-2 font-bold tracking-tight active:scale-95`
- **Form Inputs:** `rounded-xl h-11 border-2 border-border-subtle bg-white font-bold tracking-tight`
- **Icons:** Consistent `stroke-width: 2.5` for a clean, professional look.

## Migration Status
- **Compact UI Refinement:** Complete. Switched to Inter, reduced padding, and standardized font scale.
- **Bento Core:** Complete. 12-column responsive grid remains the layout foundation.
